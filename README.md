# pyTweetBot

## Find new tweets

`python pyTweetBot find-tweets --config /etc/bots/bot.conf --model /etc/bots/models/find_tweets.p --features words+bigrams+trigram`

## Find new retweets

`python pyTweetBot find-retweets`
